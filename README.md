# GhostWall
# A HenHacks Submission by Aidan L, Lathe E, Nicholas C, Andrew X
**GhostWall** is a **port-migration honeypot solution** for cybersecurity professionals, designed to allow for the **continuous improvement of cyber-attach countermeasures** while providing **secure uptime** for your service. **GhostWall** can be used by anyone, and bridges the gap between professional and beginner technicians with smart alerts, security threat explanations, and an easy-to-use interface. 

If you value your security _and_ uptime, trust **GhostWall**.
